### Automatic Deployment

1.  Upload all files to your GitHub repository.
2.  The included `.github/workflows/deploy.yml` file will automatically trigger an Action.
3.  Go to the **Actions** tab in your repository to see the build progress.
4.  Once the "Build and Deploy" action is green (Success):
    *   Go to **Settings** > **Pages**.
    *   Under **Build and deployment** > **Source**, select **Deploy from a branch**.
    *   Under **Branch**, select **`gh-pages`** (This branch is created automatically by the script).
    *   Click **Save**.

Your site will be live at `https://your-username.github.io/your-repo-name/`
